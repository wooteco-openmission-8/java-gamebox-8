# 🧱 TETRIS

## 🧩 게임 소개
**Tetris Game**은 **Java**로 구현된 고전 테트리스 게임이다.  
플레이어는 낙하하는 블록을 회전시키고 이동시켜 **가로줄을 완성**해야 한다.  
가로 한 줄이 모두 채워지면 해당 줄이 제거되고 **점수가 증가**한다.  
게임이 진행될수록 블록의 낙하 속도가 빨라지며,  
더 빠른 **판단력과 반응 속도**가 요구된다.

## 🎮 조작법 (Key Controls)

|    키     | 기능 설명 |
|:--------:|:-----------|
|  ⬅️ `←`  | 블록을 왼쪽으로 이동 |
|  ➡️ `→`  | 블록을 오른쪽으로 이동 |
|  ⬇️ `↓`  | 블록을 빠르게 내리기 *(Soft Drop)* |
|  ⬆️ `↑`  | 블록 회전 *(Rotate)* |
| `Space`  | 즉시 바닥으로 내리기 *(Hard Drop)* |
| `P`, `p` | 게임 일시정지 / 재개 |
| `R`, `r` | 게임 재시작 |
| `Q`, `q`  | 게임 종료 |

## ⚙️ 게임 규칙 (Rules)

### 🎲 기본 설정
- 한 블록의 크기는 **64px**이다.
- 보드는 **가로 10칸 × 세로 20칸**으로 구성되어 있다.
- 블록은 항상 **보드 상단 중앙**에서 스폰된다.
- 스폰 시 다른 블록과 충돌하면 즉시 **게임 오버**가 된다.

---

### 🧱 블록 규칙
- 블록의 종류는 **I, O, T, S, Z, J, L** 총 7가지이다.
- 블록은 **랜덤 생성 시스템**에 따라 무작위로 등장한다.
- 블록은 기본적으로 **시계 방향**으로 회전하며,  
  필요 시 **벽 근처에서도 회전이 가능**하다.
- 블록이 바닥이나 다른 블록에 닿으면 즉시 고정되지 않고  
  잠시의 **딜레이(Lock Delay)** 동안 추가 이동이 가능하다.
- O블록은 회전하지 않는다.

---

### 💥 라인 및 레벨
- 레벨은 1 - 10레벨까지 존재한다.
- 한 줄이 가득 차면 해당 줄이 제거된다.
- 한 줄이 제거되면 위의 블록들이 아래로 떨어진다.
- 여러 줄이 동시에 채워질 경우 **콤보로 한 번에 제거**된다.
- **10줄을 제거할 때마다 레벨이 1 상승**한다.
- 다음 레벨까지 필요한 라인 수는 **GOAL**로 표시된다.
- 레벨이 올라 갈수록 스피드가 증가한다. 

---

### 💯 점수 시스템
- 점수는 제거한 줄의 수에 따라 다르게 부여된다.
    - 1줄 제거: **100점**
    - 2줄 제거: **300점**
    - 3줄 제거: **500점**
    - 4줄 제거: **800점**
    - 5줄 이상 제거 시: **(줄 수 × 300점)** 으로 계산된다.

---

### 🧩 UI 및 표시 요소
- 화면에는 다음 정보가 표시된다:
    - **고스트 피스(Ghost Piece)** : 예상 낙하지점
    - **다음 블록 큐(Next)** : 다음에 등장할 블록
    - **현재 레벨(Level)** : 진행 중인 레벨
    - **점수(SCORE)** : 현재 점수
    - **목표 라인 수(GOAL)** : 다음 레벨까지 필요한 라인 수

---

### 🎮 조작 및 기타 규칙

- **R 키** : 게임 재시작
- **Q 키** : 게임 종료
